# Founder Photos

Place founder photos in this folder with the following filenames:

| Founder | Filename |
|---------|----------|
| Prof. Pingchuan Ma | `pingchuan.jpg` |
| Prof. Shuai Wang | `shuai.jpg` |
| Dr. Zongjie Li | `zongjie.jpg` |

## Requirements

- **Format**: JPG or PNG
- **Size**: Recommended 400x400px or larger (will be displayed as 96x96px circle)
- **Aspect Ratio**: 1:1 (square)
- **Style**: Professional headshot preferred

## Fallback

If no photo is provided, the website will automatically display a colored avatar with the founder's initials:
- PM (Pingchuan Ma) - Blue gradient
- SW (Shuai Wang) - Purple gradient
- ZL (Zongjie Li) - Green gradient
